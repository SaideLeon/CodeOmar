# **App Name**: Word Assistant

## Core Features:

- Definition Lookup: Users can enter a word and receive its definition from an online dictionary API.
- Synonym Suggestion: Provides a list of synonyms for a given word using a thesaurus API.
- Sentence Generation Tool: Generates example sentences for a given word using an AI model to demonstrate usage in context.
- Pronunciation Guide: Offers an audio pronunciation of the word, if available, or displays a phonetic transcription.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) for a calm, intellectual feel.
- Background color: Light grey (#F0F4F8) for a clean and readable interface.
- Accent color: Pale yellow (#FFECB3) to highlight important information and CTAs.
- Body and headline font: 'Literata' for a readable, slightly vintage style.
- Use simple, line-based icons for word-related actions (search, play, save).
- Prioritize a clean, single-column layout for easy content consumption.
- Subtle animations on word lookup and when new content appears.
